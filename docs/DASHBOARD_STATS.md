# Dashboard and stats

The dashboard provides:

- Total spending for a selected year
- Monthly spending chart (JOD)
- Electricity energy charts:
  - import kWh
  - export kWh
  - net kWh
  - billed kWh
- Latest bills list
- Filters:
  - utility type
  - year
  - meter id

All queries are scoped to the authenticated user.
